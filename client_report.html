Client Report

Content/Images

I decided to use the content and images based off climbing which is my favorite hobby. I chose images that looked really nice aesthetically and would look bold on a website. For the content, I just picked good stories and info that would be relevant for a climbing website.

Developer Communication

I mainly communicated with my developer over text. This is how we figured out questions and issues and we sent files for the website primarily over email. There were no communication issues and everything went very well.

Developer Expectations

My developer met and exceeded site expectations. I love how it turned out!

Future Developers

Next time I have someone develop a project for me, I would double check all wireframes to ensure they don't have any vague or empty info that needs to be corrected.

Developer Report

Communication with Client

I communicated with my client exclusively over email. We shared files and updates with eachother in a good timely manner and there were no issues.

Publishing and Sharing

Every time I made changes during a programming session, I would push them to GitHub when I was done and then promptly email my client to let him know what I did with the current update.

Managing and GitHub

Managing the site with GitHub went surprisingly well. Using the desktop app, I had no trouble pushing my code and getting updates out on time.

Experience and Reflections

Challenges

I would say that the most challenging part of the project for me was designing my own site. The site I developed for my client was straightforward and fun, but designing my own site was actually more difficult. I'm very picky about creative manners, so designing my own site included a lot of revising. I obviously overcame it and the design worked well.

Enjoyment

The thing I enjoyed the most was seeing the progress of making my client's site. Seeing each step of progress in a project like this feels super rewarding, and I'm incredibly happy with how the site ended up turning out.

Differences I Would Make

If I were to do this project again, I would do 2 things differently. first, I would give my developer images that were already optimized to remove any headache for them. I would also get a bit more creative with the interactive parts of CSS to make my client's site a bit more fun.

Confidence

I definitely feel much, much more confident on working with real world projects after this assignment. Working with both client side and developer side was really good exposure for all aspects of web development. Updating and asking clients and developers questions along the development process was also good practice for fighting against things like scope creep. Overall, I feel much more confident with all this hands on experience.
